การช่วยเหลือระหว่างติดตั้งระบบตรวจสอบการเข้าใช้คอมพิวเตอร์

เชื่อมต่อไปยังฐานข้อมูล

การช่วยเหลือ: โฮสต์ฐานข้อมูล

* ข้อมูลในช่องเป็นข้อมูลเริ่มต้นสำหรับการใช้งานทั่ว ๆ ไป

* ถ้าคุณกำลังติดตั้งบนเซิร์ฟเวอร์ Windows และกำลังใช้ MySQL การใช้ "localhost" อาจไม่สามารถใช้ได้สำหรับชื่อเซิร์ฟเวอร์ ถ้าไม่สามารถใช้ได้ ให้ลองใช้ "127.0.0.1" สำหรับที่อยู่ IP เฉพาะที่



การช่วยเหลือ: ชื่อผู้ใช้ฐานข้อมูล

* ข้อมูลในช่องเป็นข้อมูลเริ่มต้นสำหรับการใช้งานทั่ว ๆ ไป

* ป้อนชื่อผู้ใช้ที่จะใช้เชื่อมต่อไปยังฐานข้อมูลในระหว่างกระบวนการติดตั้ง ซึ่งเป็นชื่อผู้ใช้สำหรับฐานข้อมูลของคุณ



การช่วยเหลือ: รหัสฐานข้อมูล

* ข้อมูลในช่องเป็นข้อมูลเริ่มต้นสำหรับการใช้งานทั่ว ๆ ไป

* ป้อนรหัสผ่านที่จะใช้เชื่อมต่อไปยังฐานข้อมูลในระหว่างกระบวนการติดตั้ง ซึ่งเป็นรหัสผ่านสำหรับฐานข้อมูลของคุณ

การช่วยเหลือ: ชื่อผู้ใช้ของคุณ

* ป้อนชื่อผู้ใช้ที่ต้องการของคุณที่นี่ ตัวอย่างเช่น "admin"

* ชื่อนี้จะเป็นชื่อที่คุณจะใช้สำหรับเข้าสู่ระบบฐานข้อมูลการเช็คชื่อนักเรียน
